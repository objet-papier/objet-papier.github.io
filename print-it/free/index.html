Pour ce nouveau numéro, et pour tâcher de célébrer dignement les 30 ans du World Wide Web Une toile d’araignée tissée par Tim Berners-Lee en 1989 à Genève. , nous avons demandé à nos ingénieurs de porter cette idée à son paroxysme.

Outils de conception web

On prend sa respiration et on plonge tout de suite dans l’océan obscur et vertigineux des coulisses de cette édition. N’oublie pas ta lampe frontale, la connaissance est au bout du périple !

Atom

Atom est un éditeur de texte pour mac-os, linux
et Windows. Il permet d'écrire du code source.
Atom est l'outil de création de la structure HTML, des feuilles CSS et du Javascript Découvre le détail de ces acronymes à la page 10. qui compose Print-it. Il lui a permis petit à petit de prendre forme.

Jekyll

Jekyll est un générateur de site statique dévéloppé par Tom Preston-Werner, écrit en
langage de programmation Ruby. Sympa comme tout, Jekyll est sympa mais qu’en est-il de Mr Hyde ? :o il est considéré comme un des générateurs de sites statiques les plus populaires.
Jekyll s'est occupé de synchroniser les différentes parties du code (HTML/CSS/JS…) de Print-it.

GitHub

GitHub est une plateforme d’hébergement et de gestion de codes sources basée sur le protocole de versionning Git et sur les méthodes de travail collaboratif propre à l’open source.
Github héberge le code source de Print-it et permet de regrouper et synchroniser les changements qui lui sont apportés.

Siteleaf

Développé par Oak Studio, Siteleaf est le compagnon idéal de Jekyll. Il se connecte à GitHub et héberge le site sur une URL. Son CMS facilite la mise à jour du contenu du site.
Siteleaf a permis de synchroniser toutes les modifications apportées à Print-it via le code et via le CMS puis d'afficher le résultat sur une URL.

Chrome

Navigateur web développé par Google Cette petite entreprise familiale nous a supplié de leur faire de la pub, on a pas su dire non :/ c’est un logiciel conçu pour accéder aux URL et afficher le contenu des pages du World Wide Web. Le terme «navigateur» est inspiré de
Netscape Navigator, un navigateur phare apparu au milieu des années 1990. Les navigateurs Web les plus utilisés à l’heure actuelle sont Google Chrome, Mozilla Firefox,
Microsoft Edge et Apple Safari.
C’est grâce à Chrome que tu peux visualiser l’interface d’accueil de Print-it rien qu’avec tes yeux. On te recommande l'utilisation de celui-ci pour l’impression car il est le seul et l’unique navigateur à gérer le CSS print comme un chef !

Du web au print

HTML

HyperText Markup Language

Le HTML est un langage de description de données structuré en balises, conçu pour décrire le contenu d’une page Web. C’est le squelette d’une page web. Il s’agit de l’une des trois inventions, avec HTTP et URL qui ont permis l’existence du Web.

CSS

Cascading Style Sheets

Les bas-fonds d’Internet sont remplis de sigles hostiles aux voyelles, on se demande bien ce qu’ils en ont fait. Les feuilles de style en cascade (en français) désignent un langage informatique servant à décrire la présentation de documents Web (généralement des pages Web). C’est le style ou l’apparence d’une page web.

CSS Print

Le CSS Print est un ensemble d’instructions écrites en langage informatique CSS servant à gérer l’aspect imprimé d’une page Web. Il est ainsi possible d’utiliser le Web comme un environnement de mise en page et de publication multi-supports.

PDF

Portable Document Format

On s’accroche c’est bientôt l’heure de s’amuser ! Plus connu sous son acronyme, le PDF est un langage de description de page présenté par la société Adobe Systems en 1992 et qui est devenu une norme ISO en 2008. La spécificité de celui-ci est de préserver la mise en page d’un document telle qu'elle a été définie par son auteur, et cela quel que soit le logiciel, le système d'exploitation ou l'ordinateur utilisé pour l’imprimer ou le visualiser.
10

Lexique

Code source Le code source d’un programme informatique est un texte contenant des instructions rédigées dans un ou plusieurs langages de programmation, il est lu par la machine qui exécute ces instructions.
CMS (Content Management System) C’est un type de programme permettant de modifier le contenu d'un site web via une interface graphique.
Site statique Un site statique génère la même page pour tous les internautes à l’inverse d’un site dynamique qui variera son contenu en fonction de l’internaute.
Langage de programmation C’est un système destiné à l’écriture du code source. Comme les langues naturelles, chaque langage de programmation possède un alphabet, une sémantique, un vocabulaire et des règles syntaxiques spécifiques.
URL (Uniform Resource Locator) Désigne l’adresse Web permettant à un navigateur Web de localiser une ressource en ligne (fichier, image, page Web, etc.). Formalisé par Tim Berners-Lee en 1989, le protocole permettant aux URL de fonctionner est une des inventions à la base du World Wide Web.
HTTP (Hypertext Transfer Protocol) C'est le moyen de communication entre le serveur et le navigateur. HTTPS est la variante Sécurisée du HTTP.
Open source En opposition à la philosophie du logiciel propriétaire, l’open source est une méthodologie de développement pragmatique basée sur le travail collaboratif et sur le partage du codes sources.
Versionning (Gestion de version en français) La gestion de versions consiste à maintenir l'ensemble des modifications d'un ou plusieurs fichiers par une ou plusieurs personnes. Essentiellement utilisée dans le domaine de la création de logiciels, elle concerne surtout la gestion des codes source.

GeoCities, c'est quoi ?
Novembre 1994, la société Beverly Hills Internet propose quinze mégaoctets d'hébergement gratuit sur le web. Si cet espace est dérisoire à l'heure actuelle et n'accueille guère plus que quelques images, c'est une luxuriante promesse qui attire des dizaines de milliers d'utilisateurs dès ses premiers mois d'existence. Rapidement rebaptisé GeoCities le simple Quelle histoire ! service d'hébergement se transforme en outil de création de site internet,
et intègre un «Content Management System» permettant l'édition de contenu via une interface graphique. Si le CMS GeoCities demande quelques connaissances dans ses premières versions pour créer en toute liberté, il évolue vers une expérience accessible permettant aux utilisateurs les moins expérimentés de se faire une place dans cet univers. À la fin des années 90, Internet en est encore à ses balbutiements et convaincre les utilisateurs demande plus qu'une interface ergonomique. Pour répondre à ce besoin, les créateurs de GéoCities introduisent une dimension «skeuomorphique» à leur produit, et mime l'emménagement d'un espace. Lors de la création d'une page, on s'y installe comme dans la vie réelle, on choisit une ville,
un quartier puis on se voit attribué un numéro,
le tout définissant l'adresse de notre nouvelle résidence virtuelle, où l'on peut façonner son coin d’Internet et le partager avec le Monde. GeoCities est un lieu d’expression libre ou les blogs d’adolescents avoisinent les récits de vétérans de guerre et les fansites de popstars japonaises. C’est ce qu’on appelle la convergence des luttes.
Ces sujets divers et variés prennent vie à travers une esthétique marquée par gifs, typographies et word-arts en «Web Safe Colors» (216 couleurs). Print-it revient sur cet internet du passé et te donne les clés essentielles d'une page GeoCities digne de ce nom.
Les essentiels d’une page GeoCities
  • Fonds en mosaïque Les images d'arrière plan de 1000 pixels de large ne sont pas les meilleures amies du 56k. En 1994 les fonds sont des répétitions d'une image de petite taille par soucis de bande passante.

  • «Microbars» ou «timbres» Petites images d'une taille spécifique servant à revendiquer une opinion sur l'utilisation d'un navigateur plutôt qu'un autre ; ou simplement clamer son amour pour les chats.
  • Gifs animés Ce format permet d'afficher des animations dans des fichiers très légers, il a fait fureur bien avant snapchat et les réseaux sociaux, surtout sur GeoCities.
  • Mention «en construction» Une page GeoCities est une œuvre d'art en constante évolution, il ne faut surtout pas oublier de le préciser, pour faire revenir votre visiteur.
  • Typographies diverses, souvent en format image L'Internet des années 90 se limitait aux quelques fontes de base, les typographies spécifiques devaient être insérées dans des images.
  • «Hotlinks» Liens qui utilisent une resource d'un autre site, une image le plus souvent. Cause de beaucoup de lien morts sur GeoCities. Le terme a souvent été utilisé pour désigner les liens des sites GeoCities, à tort.
  • «Frames» Un frame est une sous fenêtre contenant une autre page html avec son défilement propre. Les frames ont permis des designs plus poussés et des expériences parfois désastreuses pour l'utilisateur.
  • Mise en page fixe En 1994, l'affichage web est normée par une taille, la consultation s'effectuant le plus souvent sur un moniteur affichant une résolution de 800 pixels de large par 600 de haut.
Références à GeoCities
Métropole virtuelle autrefois surpeuplée et en perpétuelle construction, ses quartiers sont aujourd'hui déserts, les habitants ayant migré vers un nouvel eldorado porté sur des interactions plus poussées entre les utilisateurs : les réseaux sociaux. Faisant place à la navigation «monitoré» et à l'internet marketing, l'expérience GeoCities a de quoi rendre les internautes des années 90 nostalgiques. Nous avons sélectionné quelques projets lui rendant hommage, à découvrir ci-contre.

Cameron's world
Véritable lettre d'amour à GeoCities, Cameron's world est l'œuvre du designer néo zélandais Cameron Askin. À travers une page regorgeant de visuels tout droit sortie de la blogosphère de 1996, il nous transporte dans différents quartiers, l'expérience est saisissante, presque plus vraie que nature.

Deleted City
Quinze ans d'activité et 650 Go d'archives, GeoCities est un sacré pavé d'internet, et pas des plus accueillant à explorer. Richard Vijgen nous facilite la tâche avec Deleted City, une carte interactive de l'archive de GeoCities, de quoi visualiser l'intégralité de son contenu et se rendre compte de ses proportions.

GeoCities-izer
Après avoir parcouru cet article vous ne rêvez que d'une chose : qu'internet tout entier ressemble à GeoCities. Grâce à GeoCities-izer ce rêve devient réalité, il vous suffit d'entrer l'adresse de votre site préféré afin de le transformer en une version «GeoCities ready», pour ne plus se prendre la tête avec l'ergonomie et tout ce bazar.

NeoCities
L'esthétique' des sites geocities peut paraitre daté pour beaucoup d'entre nous, les modes se renouvelles. Ce sont plusieurs milliers d'utilisateurs qui ont créé des versions modèrnes de sites GeoCities, en concervant cette patte graphique si particulière. NeoCities regroupe le travail de tous ces fondus de Netscape.

Les bonnes choses ont une fin
Si le service a fermé ses portes en 2009,
The Archive Team a sauvé les 650 Go avant la suppression définitive, l'archive est consultable librement Une enquête dûment menée par Corentin Moussard. à différents endroits sur le web. La version japonaise de GeoCities quand a elle aura subsisté une décennie de plus, elle fermera ses portes en mars 2019.
