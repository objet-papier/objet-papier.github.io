Outils de conception web

On prend sa respiration et on plonge tout de suite dans l’océan obscur et vertigineux des coulisses de cette édition. N’oublie pas ta lampe frontale, la connaissance est au bout du périple !

Atom

Atom est un éditeur de texte pour
mac-os, linux et Windows. Il permet d'écrire du code source.
Atom est l'outil de création de la structure HTML, des feuilles CSS et du Javascript Découvre le détail de ces acronymes à la page 10 qui compose Print-it. Il lui a permis petit à petit de prendre forme.

Jekyll

Jekyll est un générateur de site statique dévéloppé par
Tom Preston-Werner, écrit en langage de programmation Ruby. Sympa comme tout Jekyll est sympa mais qu’en est-il de Mr Hyde ? :o , il est considéré comme un des générateurs de sites statiques les plus populaires.
Jekyll s'est occupé de synchroniser les différentes parties du code (HTML/CSS/JS…) de Print-it.

GitHub

GitHub est une plateforme d’hébergement et de gestion de codes sources basée sur le protocole de versionning Git et sur les méthodes de travail collaboratif propre à l’open source.
Github héberge le code source de Print-it et permet de regrouper et synchroniser les changements qui lui sont apportés.
