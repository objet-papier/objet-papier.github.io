Siteleaf

Développé par Oak Studio, Siteleaf est le compagnon idéal de Jekyll. Il se connecte à GitHub et héberge le site sur une URL. Son CMS facilite la mise à jour du contenu du site.
Siteleaf a permis de synchroniser toutes les modifications apportées à Print-it via le code et via le CMS puis d'afficher le résultat sur une URL.

Netscape Navigator, un navigateur phare apparu au milieu des années 1990. Les navigateurs Web les plus utilisés à l’heure actuelle sont
Google Chrome, Mozilla Firefox,
Microsoft Edge et Apple Safari.
