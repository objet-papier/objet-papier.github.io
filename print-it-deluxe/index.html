Chez Objet Papier, vous le savez, nous n’aimons rien de moins que d’explorer les liens qui existent entre le physique et le numérique, entre le fabuleux univers virtuel qui offre tant de possibilités et la noblesse de l’objet palpable, façonné par la main humaine, fruit de traditions parfois millénaires. Atom

Développé par l'équipe de GitHub, Atom est un éditeur de texte pour
mac-os, linux et Windows. Il permet d'écrire du code source.
Il est basé sur Chromium et Electron.
Atom a permit l'édition de la structure HTML, des feuilles CSS et du Javascript qui font Print-it !

Jekyll

Jekyll est un générateur de site statique dévéloppé par
Tom Preston-Werner, écrit en langage de programmation Ruby. Il est considéré comme le générateur de site statique le plus populaire.
Jekyll s'est occupé de synchroniser les différentes parties du code de Print-it grâce à un langage de templating.

GitHub

Lancé en 2008, GitHub est une plateforme d’hébergement et de gestion de codes sources basée sur le protocole de versionning (versionnage) Git et sur les méthodes de travail collaboratif propre à l’open source.
Github héberge le code source de Print-it et permet de synchroniser les changements qui lui sont apportés.

Les outils de conception web

Siteleaf

Produit développé par Oak Studio, Siteleaf est le compagnon idéal du générateur de site statique. Il permet de se connecter à GitHub afin de récupérer le code source et d’en héberger la version statique. Il y ajoute un CMS facilitant l’édition du contenu du site.
Siteleaf nous à permis de synchoniser toutes les modifications apportées à Print-it via le code et via le CMS et d’afficher le résultat sur une URL.

Chrome

Navigateur web développé par Google basé sur la technologie Webkit il permet d'accéder aux URL et d'afficher le contenu des pages du World Wide Web. Le terme « navigateur » est inspiré de
Netscape Navigator, le navigateur phare apparu au milieu des années 1990. Les navigateurs Web les plus utilisés à l’heure actuelle sont
Google Chrome, Mozilla Firefox,
Microsoft Edge et Apple Safari.
Nous recommandons l'utilisation de Chrome pour imprimer Print-it, car il est le seul navigateur à gérer le CSS print de manière aboutie.

HTML

Acronyme d’HyperText Markup Language, le HTML est un langage de description de données structuré en balises, conçu pour décrire le contenu d’une page Web. Il s’agit de l’une des trois inventions, avec HTTP et URL qui ont permis l’existence du Web.

CSS

Les feuilles de style en cascade (CSS, de l’anglais Cascading Style Sheets) désignent un langage informatique servant à décrire la présentation de document Web (généralement des pages Web). Le CSS Print est un ensemble d’instructions écrites en langage informatique CSS* servant à gérer l’aspect imprimé d’une page Web. Il est ainsi possible d’utiliser le Web comme un environnement de mise en page et de publication multi-supports.

PDF

Le Portable Document Format, communément abrégé en PDF, est un langage de description de page présenté par la société Adobe Systems en 1992 et qui est devenu une norme ISO en 2008. La spécificité du PDF est de préserver la mise en page d’un document telle qu'elle a été définie par son auteur, et cela quels que soient le logiciel, le système d'exploitation et l'ordinateur utilisés pour l’imprimer ou le visualiser.

GeoCities, l'internet du passé
GeoCities, l'internet du passé

19
Novembre 1994, Beverly Hills Internet propose quinze mégaoctets d'hébergement gratuit sur le web. Si cet espace est dérisoire à l'heure actuelle et n'accueille guère plus que quelques images, c'est une luxuriante promesse qui attire des dizaines de milliers d'utilisateurs dès les premiers mois d'existence de l'entreprise. Rapidement rebaptisé GeoCities le simple service d'hébergement se transforme en outil de création de site internet, et intègre un "Content management system" permettant l'édition de contenu via une interface graphique. Si le CMS GeoCities demande quelques connaissances dans ses premières versions pour créer en toute liberté, il évolue vers une expérience accessible permettant aux utilisateurs les moins expérimentés de se faire une place dans cet univers. À la fin des années 90, internet en est encore à ses balbutiements et l'acquisition d'utilisateurs demande plus qu'une interface ergonomique. Pour répondre à ce besoin, les créateurs de GéoCities introduisent une dimension "skeuomorphique" à leur produit, et mime l'emménagement d'un espace. Lors de la création d'une page, on s'y installe comme dans la vie réelle, on choisit une ville, un quartier puis on se voit attribué un numéro, le tout définissant l'adresse de notre nouvelle résidence virtuelle, où l'on peut façonner son coin d’internet et de le partager avec le monde. GeoCities est un lieu d’expression libre ou les blogs d’adolescents avoisinent les récits de vétérans de guerre et les fansites de popstars japonaises. Ces sujets divers et variés prennent vie à travers une esthétique marqué par gifs, typographies et word-arts en « Web Safe Colors » (216 couleurs). Print-it revient sur cet internet du passé dans une analyse des éléments essentiels d'un page GéoCities digne de ce nom.

Métropole virtuelle autrefois surpeuplé en perpétuelle construction, ses quartiers sont aujourd'hui déserts, les habitants ayant migré vers un nouvel el-dorado porté sur des intéractions plus poussées entre les utilisateurs : les réseaux sociaux. Faisant place à la navigation « monitoré » et à l'internet marketing, l'expérience GeoCities a de quoi rendre les internautes des années 90 nostalgiques. Après avoir écumé les articles et sites sur le sujet nous avons sélectionné quelques projets en hommages et références à GeoCities.
Références à GeoCities
- Cameron's world
Véritable lettre d'amour à GeoCities, Cameron's world est l'œuvre du designer néo zélandais Cameron Askin. À travers une page regorgeant de visuels tout droit sortie de la blogosphère de 1996, il nous transporte dans différents quartiers, l'expérience est saisissante, presque plus vrai que nature.

- Deleted City
Quinze ans d'activité et 650Go d'archives, GeoCities est un sacré pavé d'internet, et pas des plus accueillant à explorer. Richard Vijgen nous facilite la tâche à travers Deleted City, une carte interactive de l'archive de GeoCities, de quoi visualiser l'intégralité de son contenu et ses proportions.

- GeoCities-izer
Après avoir parcouru cet article vous ne rêvez que d'une chose : qu'internet tout entier soit comme GeoCities. Grâce à GeoCities-izer ce rêve devient réalité, il vous suffit d'entrer l'adresse de votre site préféré afin de le transformer en une version "GeoCities ready", pour ne plus se prendre la tête avec l'ergonomie et tout ce bazar.
