Atom

Développé par l'équipe de GitHub, Atom est un éditeur de texte pour
mac-os, linux et Windows. Il permet d'écrire du code source.
Il est basé sur Chromium et Electron.
Atom a permit l'édition de la structure HTML, des feuilles CSS et du Javascript qui font Print-it !

Jekyll

Jekyll est un générateur de site statique dévéloppé par
Tom Preston-Werner, écrit en langage de programmation Ruby. Il est considéré comme le générateur de site statique le plus populaire.
Jekyll s'est occupé de synchroniser les différentes parties du code de Print-it grâce à un langage de templating.

GitHub

Lancé en 2008, GitHub est une plateforme d’hébergement et de gestion de codes sources basée sur le protocole de versionning (versionnage) Git et sur les méthodes de travail collaboratif propre à l’open source.
Github héberge le code source de Print-it et permet de synchroniser les changements qui lui sont apportés.
